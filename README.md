# 咪咕视频网络测试工具

一个用于测试咪咕视频定向流量的 Python 脚本工具。

## ⚠️ 免责声明

本工具仅用于**合法的网络测试目的**。使用本工具前请确保：
- 遵守当地法律法规
- 遵守运营商服务条款
- 仅用于测试自己的网络连接
- 不用于任何非法目的

## 功能特性

- ✅ 多线程并发请求
- ✅ 实时流量统计
- ✅ 可配置请求间隔
- ✅ 可设置测试时长
- ✅ 友好的命令行界面
- ✅ 流量和速度显示

## 环境要求

- Python 3.6+
- requests 库

## 安装

1. 克隆仓库：
```bash
git clone https://github.com/1298786483/migu-network-tester.git
cd migu-network-tester
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

### 基本使用

```bash
python migu_tester.py
```

### 高级选项

```bash
# 使用 5 个并发线程
python migu_tester.py -t 5

# 每次请求间隔 1 秒
python migu_tester.py -i 1

# 运行 60 秒后自动停止
python migu_tester.py -d 60

# 组合使用：3 个线程，间隔 0.5 秒，运行 120 秒
python migu_tester.py -t 3 -i 0.5 -d 120
```

### 参数说明

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--threads` | `-t` | 并发线程数 | 1 |
| `--interval` | `-i` | 请求间隔(秒) | 0 |
| `--duration` | `-d` | 测试时长(秒) | 0 (无限制) |

### 停止测试

按 `Ctrl + C` 可随时停止测试并查看统计信息。
